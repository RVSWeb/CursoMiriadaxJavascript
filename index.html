Contenido del curso

Este curso es muy básico lo cual solo busca una pequeña introducción a las tecnologías anteriormente nombradas , muchos de los ejemplos se han sacado de fuentes externas para hacerse más claro para el lector lo que se intenta exponer ya que el curso en si estaba falto de contenido , ejemplos claros y demás elementos

Espero que entienda el esfuerzo que he realizado para poder hacer más entendible los expuestos a los usuarios que visiten la pagina , esto no pretende ser un manual o tutorial a lo sumo sino una exposición de contenido de un curso ya realizado donde la gente que lo desee pueda visitarlo sin compromiso ninguno para adquirir pequeñas pinceladas de estas tecnologías

Con el tiempo se iran añadiendo nuevos contenidos , mejorando otros , corrigiendo errores y demás temas referentes a los contenidos que se manejan , si por algun motivo hay erratas , problemas o fallos , comprendan que es normal que suceda cuando se empieza en el mundo de la tecnología y más en estos campos , por eso les pido un poco de comprensión

Javascript

Lenguaje de programación interpretado.
Orientado a objetos basado en prototipos,
Imperativo, débilmente tipado y dinámico.

JQuery

jQuery es una biblioteca de JavaScript
Permite interactuar los documentos HTML
Manipular el árbol DOM, manejar eventos y usar la técnica AJAX

Ajax

Ajax es una técnica de desarrollo web
para crear aplicaciones interactivas o RIA (Rich Internet Applications)
Mantiene la comunicación asíncrona con el servidor en segundo plano.

AngularJS

AngularJS es un framework de Javascript
Se utiliza para crear y mantener aplicaciones
Web de una sola página (SPA).

NodeJS

NodeJS es un entorno en tiempo de ejecución
multiplataforma , asíncrono , de código abierto
para la capa del servidor basado en el lenguaje de programación ES6.
