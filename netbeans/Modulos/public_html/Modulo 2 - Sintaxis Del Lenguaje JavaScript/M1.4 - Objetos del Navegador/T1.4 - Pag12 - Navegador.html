Objetos del Navegador

Cuando se carga una página en un navegador se crean un número de objetos caracteríticos del Navegador

window object
location object history object document object navigator object screen object
form object images object link object
 

Los ejemplos se ven a través de la consola del navegador

Objeto Window

Contiene propiedades de la ventana y trabaja con 'frames'

Se genera un objeto window para cada uno

Objecto document contiene todas la propiedades del documento actual

  • color
  • fondo
  • enlaces
  • imágenes
Ventana Navegador o 'Marco - frame' representa objeto window
Pagina HTML se representa mediante
objeto document
